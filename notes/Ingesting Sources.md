# Ingesting Sources

The filing convention for this wiki. For each source worth keeping, the agent creates one note per *idea* (not per source), tags it, links it to existing notes with [[Wikilinks]], and adds it to the relevant hub.

Conventions: frontmatter `tags:`, a `Source:` line with the URL, and at least two links so nothing enters the graph as an orphan. See [[Research Workflow]], [[Source Triage]].
