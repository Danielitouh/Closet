# Research Workflow

The loop this repo automates: **discover** ([[Exa Search]]) → **collect** ([[Web Reading]], [[YouTube Transcripts]], [[RSS Feeds]], [[Twitter Research]]) → **distill** ([[Progressive Summarization]]) → **file** ([[Ingesting Sources]]) → **review** ([[Weekly Review]]).

Run it by telling Claude: *"Research X across the web and add what matters to my wiki."*
