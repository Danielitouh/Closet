# Agent Reach

The internet-access layer of this repo: a skill that routes Claude Code to 15 platforms through the currently-best backend for each. Run `agent-reach doctor` to see live status.

Zero-config channels: [[Web Reading]], [[Exa Search]], [[YouTube Transcripts]], [[Bilibili Research]], [[V2EX]], [[RSS Feeds]]. Credential-gated: [[Twitter Research]], [[Reddit Research]], and more.

Combined with this wiki it closes the loop: reach → read → file. See [[Ingesting Sources]] and [[Research Workflow]].
