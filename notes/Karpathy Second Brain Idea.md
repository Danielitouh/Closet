# Karpathy Second Brain Idea

Karpathy's viral suggestion: stop using AI only to write code — point [[Claude Code]] at a folder and let it build and maintain a personal wiki. Drop in a source, the agent reads, links, and files it.

This repo is an implementation: [[Agent Reach]] fetches, the agent writes [[Atomic Notes]] with [[Wikilinks]], and [[Graph View]] shows the compound interest. Workflow details: [[Ingesting Sources]].
